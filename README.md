## 📂 Projetos em Destaque

### 🚲 Sistema de Aluguel de Bicicletas
Sistema completo de gerenciamento que utiliza o padrão de arquitetura **MVC** e **DAO**.
* **Funcionalidades:** Cadastro e controle de usuários, bicicletas e estações, além do registro formal de aluguéis.
* **Persistência:** Utiliza JDBC para integração com banco de dados PostgreSQL.
* **Interface:** Desenvolvido com Java Swing.

### 🎬 Gestão de Filmes e Elenco
Aplicação para controle de acervo cinematográfico e artistas, focada na organização de dados e relacionamentos.
